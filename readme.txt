- El código HTML proporciona la estructura básica de la página web de la lista de tareas:
- El código CSS proporciona estilos para hacer que la página de la lista de tareas 
sea visualmente atractiva y funcional.
- El código JavaScript proporciona la funcionalidad dinámica para la aplicación de lista de tareas.
addTask(task): Agrega una nueva tarea a la lista con la fecha de creación.
removeTask(taskElement): Elimina una tarea de la lista.
toggleTask(taskElement): Cambia el estado de completado de una tarea.
editTask(taskElement): Permite al usuario editar una tarea existente.
saveTasksToLocalStorage(): Guarda las tareas en el almacenamiento local del navegador.
loadTasksFromLocalStorage(): Carga las tareas guardadas del almacenamiento local al cargar la página.
filterTasks(filter): Filtra las tareas según el filtro seleccionado (todas, pendientes, completadas).
searchTasks(): Busca tareas en la lista según el texto ingresado en el campo de búsqueda.
